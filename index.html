Nuclearmente

A nuclear horror-themed blog platform that allows users to log in, create, and share their own posts.

Project Description

Nuclearmente is a web application designed for users interested in nuclear horror themes. It features a secure authentication system (log in and log out) connected to a database, ensuring safe user access. The platform includes an interactive blog section, where users can create and publish posts related to nuclear horror stories, theories, and discussions.

Technologies Used

  • Frontend: HTML, CSS
  • Backend: Nest.js
  • Database: PostgreSQL

Role & Contributions

  • Designed and implemented the user authentication system (log in/log out).
  • Developed a modern and responsive UI, focusing on dark and immersive aesthetics.
  • Built a blog feature where users can share nuclear horror-themed content.
  • Integrated the frontend with the backend to ensure seamless data flow and security.
